# Local Agent Contract Summary

An OpenSciFlow local agent must only execute commands that come from reviewed command templates.

## Required Checks

- Manifest validates.
- Workflow validates.
- Required inputs exist.
- Required model weights are available or explicitly approved for retrieval.
- License and citation metadata are present.
- Hardware and environment requirements are shown.
- Command template placeholders are declared.
- Run directory is approved.

## Execution Rule

The local agent executes:

```text
reviewed template + validated inputs + approved run directory
```

It does not execute:

```text
free-form shell generated by an LLM
```
