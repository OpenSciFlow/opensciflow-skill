# Code of Conduct

OpenSciFlow Skill follows the Contributor Covenant spirit: be respectful, constructive, and evidence-focused.

## Expected Behavior

- Make corrections concrete.
- Separate evidence from opinion.
- Respect upstream maintainers and licenses.
- Avoid hype and unsupported claims.

## Unacceptable Behavior

- Harassment or personal attacks.
- Misrepresenting listed projects as partners.
- Encouraging unsafe execution of unreviewed commands.
- Presenting computational workflow demos as clinical or drug-development decisions.
