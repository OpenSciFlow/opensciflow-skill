# GROMACS RMSD Example

Mock example for:

```text
Analyze RMSD for this protein MD trajectory.
```

This example shows how an agent should:

1. match an RMSD workflow;
2. read a GROMACS manifest;
3. check required `tpr` and trajectory inputs;
4. render a reviewed command template;
5. prefer dry-run;
6. request approval before execution;
7. write a run record.

The files are examples only. They do not contain real trajectory data.
